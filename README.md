convert
=======

把中缀表达式转换成逆波兰表达式

实现思路：

1.遇到数字:直接输出（添加到后缀表达式postFix中）；

2.栈为空时，遇到运算符，直接入栈mm；

3.遇到左括号:将其入栈；

4.直到遇到右括号:执行出栈操作，并将出栈的元素存入postFix中，直到弹出栈的是左括号，左括不存出；

5.遇到其他运算符:"+，-，*，/"：弹出所有优先级大于或者等于该运算符的栈顶元素，然后将该运算符入栈；

6.最终将栈中的元素依次出栈，存入到postFix中，按格式输出。
